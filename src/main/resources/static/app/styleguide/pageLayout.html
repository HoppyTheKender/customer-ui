Page Layout

Workflow Pages

When creating a page that's part of the wizard workflow, there are classes that you can use to help make the pages stay consistent in style. Use the .select-options class for radio button options at the top. Use .selected-form for the form elements. Use .flow-nav for the navigation at the bottom of the page to get to the previous and next pages. 

<div class="col-md-12"><h2>Title of Page</h2></div>
<div class="col-md-12 select-options">
	<label><input type="radio" ng-model="$ctrl.selectedAction.action" value="..." ng-click="$ctrl.toggleSelectedActionOption()"> Radio Option 1</label><br/>
	<label><input type="radio" ng-model="$ctrl.selectedAction.action" value="..." ng-click="$ctrl.toggleSelectedActionOption()"> Radio Option 2</label><br/>
	<label><input type="radio" ng-model="$ctrl.selectedAction.action" value="..." ng-click="$ctrl.toggleSelectedActionOption()"> Radio Option 3</label><br/>
</div>
<div class="col-md-12 selected-form">
	<error-summary errors="$ctrl.errors"></error-summary>
	...
	...
</div>
<div class="col-md-12 flow-nav">
	<flow-navigation can-show-next="$ctrl.canShowNext" next="$ctrl.next()" can-show-prev="$ctrl.canShowPrev" prev="$ctrl.prev()"></flow-navigation>
</div>
